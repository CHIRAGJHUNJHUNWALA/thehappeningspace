1 What software do I need?

You only need one free tool to edit these files:

→

Download Visual Studio Code from code.visualstudio.com — it's free, works on Windows and Mac, and is the industry standard for editing websites.

→

Once installed, go to File → Open Folder → select your website folder. All your files appear in the left panel.

→

To preview, just double-click any HTML file and it opens in your browser.

Tip

Install the "Prettier" extension in VS Code — it auto-formats your code so it stays clean and readable.

2 Understanding your files

FileWhat it is
index.htmlHomepage — the first page people see
workshop-detail.htmlWorkshop enrollment page — works for ALL artists via the URL

Every file is a single HTML file. All the styling (colours, fonts, layout) and logic (buttons, payments) live inside the same file — you don't need separate CSS or JavaScript files.

3 Adding / editing workshops

All workshop data lives in workshop-detail.html. Open it in VS Code and search for const WORKSHOPS — you'll find a block that looks like this:

const WORKSHOPS = {
  'heet-samani': {
    artist: 'Heet Samani',          ← Artist name shown on page
    initials: 'HS',              ← Shows in bio avatar circle
    style: 'Hip-Hop · Breaking',   ← Dance style shown below name
    posterBg: 'linear-gradient(...)', ← Poster colour (replace with image later)
    bio: 'Artist bio text here...', ← Bio paragraph
    desc: 'Workshop description...', ← Workshop description
    packages: [                    ← Pricing options
      { id: 'p1', name: 'Single Session', desc: '1 class', price: 900, badge: null },
      { id: 'p2', name: 'Double Session', desc: '2 classes', price: 1600, badge: 'Best Value' },
    ],
    slots: [                       ← Class dates to pick from
      { id: 's1', date: 'Saturday, 7 Jun 2025', time: '11:00 AM – 1:00 PM', spots: 6 },
    ],
  },
};

To change a price

Find the artist, find their package, change the price number. Example: change price: 900 to price: 1000. Save the file.

To add a new date / slot

Add a new line inside slots: [ ]:

{ id: 's3', date: 'Saturday, 21 Jun 2025', time: '11:00 AM – 1:00 PM', spots: 20 },
Important

Every slot needs a unique id — s1, s2, s3, s4 etc. Don't repeat the same id.

To add a new package / pricing tier

Add a new line inside packages: [ ]:

{ id: 'p3', name: 'Full Intensive', desc: '3 classes + recording', price: 2200, badge: null },

Set badge to 'Best Value' or 'Popular' to show a label, or null for no label.

To add a brand new artist / workshop

1

Copy an entire existing artist block (from 'heet-samani': { to its closing },)

2

Paste it after the last artist, before the closing };

3

Change the key from 'heet-samani' to something like 'new-artist' (no spaces, lowercase, hyphens)

4

Update all the fields — artist name, style, bio, packages, slots

5

Add a card on the homepage linking to workshop-detail.html?id=new-artist

4 Editing the homepage

Open index.html. The main things you'll want to edit:

Workshop cards

Search for <!-- WORKSHOP 1 — Heet Samani -->. Each card looks like this:

<div class="workshop-card">
  <div class="card-poster-label">
    <p class="card-artist">Heet Samani</p>   ← Change name here
    <p class="card-style">Hip-Hop · Breaking</p>  ← Change style here
  </div>
  <span class="card-date">Add Date · Add Time</span>  ← Add real date
  <span class="card-price">₹XXX</span>              ← Add real price
</div>

Studio stats

Search for class="stat-num" — you'll find 4 stat boxes. Change the numbers and labels to match your real data.

Footer contact details

Search for Add WhatsApp, Add Email, Add Instagram — replace these with your real links:

<a href="https://wa.me/919876543210">WhatsApp</a>
<a href="mailto:hello@thehappeningspace.in">Email</a>
<a href="https://instagram.com/thehappeningspace">Instagram</a>

5 Adding real photos

Workshop poster images

In workshop-detail.html, find the artist you want to update and change posterBg from a gradient to an image:

// Before (placeholder gradient):
posterBg: 'linear-gradient(145deg,#1a0407,#bc1f2e)',

// After (real image):
posterBg: 'url(images/heet-samani-poster.jpg) center/cover no-repeat',

Put your images in a folder called images inside your website folder.

Photo strip on homepage

In index.html, search for class="strip-photo". Replace the placeholder divs with real images:

// Before:
<div class="strip-photo"><div class="ph" style="background:...">Add Your Photo</div></div>

// After:
<div class="strip-photo"><img src="images/studio-1.jpg" style="width:100%;height:100%;object-fit:cover"></div>
Tip — Image sizes

Poster images: 600×800px minimum. Strip photos: 560×380px minimum. Compress all images at tinypng.com before uploading — keeps the site fast.

6 Setting up Razorpay

1

Go to razorpay.com → Sign up with your studio's details

2

Complete KYC — add your bank account (this is where payments land)

3

Dashboard → Settings → API Keys → Generate Live Key

4

Copy the Key ID (starts with rzp_live_)

5

Open workshop-detail.html → find const RAZORPAY_KEY → paste your key

const RAZORPAY_KEY = 'rzp_live_YourActualKeyHere';
Never share this key

Don't post it publicly or share the file with this key filled in. Keep it private.

7 Setting up Google Sheets auto-logging

Every successful payment will automatically add a row to your Google Sheet.

1

Create a new Google Sheet. Name the columns in Row 1 exactly as:
Timestamp | Name | Phone | Workshop | Style | Package | Slots | Amount Paid | Payment ID | Order ID

2

In the sheet, go to Extensions → Apps Script

3

Delete everything in the editor and paste this code:

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  const data  = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.timestamp, data.name, data.phone,
    data.workshop,  data.style, data.package,
    data.slots,     data.amount_paid,
    data.payment_id, data.order_id,
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({ status: 'ok' }))
    .setMimeType(ContentService.MimeType.JSON);
}
4

Click Deploy → New Deployment
— Type: Web App
— Execute as: Me
— Who has access: Anyone

5

Click Deploy → copy the Web App URL

6

Open workshop-detail.html → find const SHEETS_URL → paste the URL

const SHEETS_URL = 'https://script.google.com/macros/s/YourScriptId/exec';

8 Going live on Netlify

1

Make sure your folder contains: index.html, workshop-detail.html, and your images/ folder

2

Go to netlify.com/drop

3

Drag your entire website folder onto the page

4

In 30 seconds you get a live URL like random-name-abc123.netlify.app

5

To change the URL: Netlify Dashboard → Site Settings → Change site name → set it to thehappeningspace

Custom domain (optional)

Buy thehappeningspace.in from GoDaddy (~₹800/year). In Netlify → Domain Settings → Add custom domain → follow the steps. Takes about 10 minutes.

9 Updating the site after going live

Every time you edit a file and want to update the live site:

1

Make your changes in VS Code and save the file

2

Go to your Netlify dashboard → Deploys tab

3

Drag your updated folder onto the deploy dropzone

4

Site updates in under 30 seconds — live immediately

Weekly workshop update routine

Every week: open workshop-detail.html → update slot dates → update spots count → save → drag to Netlify. Takes 5 minutes max.
